## **简单说明** ##

`./aws_mins/..` 这是一份自动站的观测记录数据，等宽；`prepare_data.R` 用来对原始数据进行分列。

`./new_data/..` 中是分列后的数据，其中每天的观测数据是单个文本文件，记录从前一天晚上的 `20:00` 开始，每分钟记录一次，至当日 `20:00` 。

数据中某些字段记录为空，可挑选有记录数据的字段来做处理。
